# Testing Scripts

Scripts para testing del MCP Server.

## Scripts Disponibles

### `test-mcp.sh`
Suite completa de tests con 18+ tests organizados en 6 suites.

**Uso:**
```bash
./scripts/test-mcp.sh
```

### `test-mcp-quick.sh`
Suite rápida con 7 tests críticos (~2 segundos).

**Uso:**
```bash
./scripts/test-mcp-quick.sh
```

### `verify-tests.sh`
Verifica que el entorno de testing esté configurado correctamente.

**Uso:**
```bash
./scripts/verify-tests.sh
```
